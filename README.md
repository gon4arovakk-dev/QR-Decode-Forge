# QR-Decode-Forge QR Decode Forge — Ридер QR-кодов на 7 языках
QR Decode Forge — это коллекция из семи независимых реализаций ридера (сканнера) QR-кодов, каждая из которых работает на своём языке программирования. Проект создан для демонстрации кроссплатформенного подхода к одной задаче, а также для предоставления удобного инструмента для декодирования QR-кодов из изображений, PDF-файлов или видеопотока с камеры.

✨ Уникальные возможности (общие для всех версий)
📷 Чтение из различных источников: статические изображения (PNG, JPG, BMP, GIF), PDF-файлы, видеопоток с веб-камеры, буфер обмена

🧩 Поддержка множества форматов: QR Code, Data Matrix, Aztec, PDF417, UPC, EAN и другие 1D/2D штрихкоды (в зависимости от библиотеки)

🎯 Автоматическое обнаружение: определение QR-кода в любом месте изображения, независимо от ориентации и масштаба

📊 Пакетное сканирование: обработка нескольких изображений или страниц PDF за раз

🔍 Расшифровка структурированных данных: автоматическое распознавание контактов (vCard), Wi-Fi, URL, email, SMS, геолокации

🖥️ Интерфейсы:

Командная строка (CLI) с гибкими опциями

Графический интерфейс (GUI) в Python, Java и C#

Веб-интерфейс с доступом к камере в JavaScript и PHP

Поддержка перетаскивания файлов (drag & drop) в веб-версиях
